## Summary
Implemented a responsive FAQ page with accordion-style questions and answers, integrated into the main navigation. Fully localized for EN, ES, FR, and RU.

## Issues Addressed / Features Added
- Created `/faq` page with "accordion" design for better readability.
- Added "FAQ" link to the main navigation bar.
- Implemented full i18n support for all 4 supported languages.
- Added a "Call to Action" at the bottom of the FAQ to drive users to the assessment.

## Implementation Details
The FAQ page uses `@headlessui/vue` Disclosure components for the accordion effect. The content is stored in the locale JSON files to ensure easy translation management.
